REVISION TKA REHAB TRACKER
===========================

HOW TO USE
----------
1. Open revision-tka-rehab.html in any modern browser (Chrome, Safari, Firefox, Edge)
2. Set your surgery date in the Settings tab
3. Start logging daily from the Today tab

HOSTING (recommended for iPhone/iPad)
--------------------------------------
Option A - Netlify Drop (easiest, 2 min):
  Go to drop.netlify.app, drag this entire folder onto the page.
  You get a live URL instantly.

Option B - Vercel:
  Go to vercel.com, drag the folder onto the dashboard.

Option C - GitHub Pages:
  Push this folder to a GitHub repo, enable Pages in repo Settings.

ADDING TO IPHONE / IPAD HOME SCREEN
-------------------------------------
1. Open the hosted URL in Safari
2. Tap the Share button (box with arrow)
3. Tap "Add to Home Screen"
4. It will work like an installed app

AI COACH TAB
------------
Requires an internet connection. All other features
(logging, checklists, timers, charts) work offline
once the page has loaded.

CROSS-DEVICE SYNC
-----------------
Data is stored locally on each device. Use Settings > Export
to save a JSON backup, then Settings > Import on another device.

FILES
-----
revision-tka-rehab.html  — the full app (open this)
manifest.json            — PWA metadata for home screen install
